# TripMakerSL-Travel Agency Management System

TripMakerSL is a Windows Forms application designed to simplify travel planning and booking. Built on .NET Framework 4.7.2, it provides an intuitive interface for users to search, select, and book transportation and hotel rooms, integrating with Oracle databases for secure and reliable data management.

## Features

- **Transport Booking:**  
  Book seats for buses, trains, and flights.  
  Real-time seat selection and availability display.  
  Multiple payment method support.

- **Hotel Room Booking:**  
  Browse available rooms by hotel.  
  View room details including amenities and pricing.  
  Select multiple rooms and confirm bookings.

- **Activity Booking:**  
  Discover and book local activities and experiences.  
  Filter activities by location and category.

- **Comprehensive Booking Management:**  
  View and manage all your bookings (transport, hotel, activity) in one place.

- **User Authentication:**  
  Secure login required for booking actions.

- **Database Integration:**  
  Uses Oracle Managed Data Access for robust data operations.

## Technologies

- C# (.NET Framework 4.7.2)
- Windows Forms (WinForms)
- Oracle Database (Oracle.ManagedDataAccess)
- Visual Studio 2022

## Getting Started

1. **Clone the repository:**
     ```bash
	git clone https://github.com/idris58/TripMaker-Travel-Agency-Management-System.git
	```

2. **Open in Visual Studio 2022.**

3. **Configure the database connection:**
- Update the connection string in `App.config` to match your Oracle database settings.

4. **Build and run the solution.**

## Usage

- Log in or register to access booking features.
- Select transport type (Bus, Train, Flight) and choose your seat.
- Browse hotels and select rooms to book.
- Discover and book activities.
- Manage and view all your bookings in one place.
- Confirm bookings and view your reservations.
